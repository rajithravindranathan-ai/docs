# Payment Details

## Introduction

The **Details** section displays key payment information for the selected order. It provides a concise view of the payment method, reference number, and card details to help users quickly verify transaction information.

---

## Payment Details – UI Reference Table

![Payment Details](/img/assets/Payment%20Details%20-%20image-20260422-070958.png)

The following table describes the item features that support common follow-up tasks.

| Callout ID | Feature | Description |
|---|---|---|
| 1 | **Method** | Display the payment method used (for example, Credit Card). |
| 2 | **Ref** (Reference Number) | Display the transaction reference identifier. |
| 3 | **Card** | Display masked card details used for the transaction. |
| 4 | **Card Type** | Display the card provider (for example, Visa). |
