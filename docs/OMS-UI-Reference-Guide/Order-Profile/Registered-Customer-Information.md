# Registered Customer Information

### Introduction

The **Customer Information** section displays key details of the **registered customer**. It provides essential identification and contact information to support order processing, communication, and verification.

---

### Customer Information UI Reference Table

![image-20260426-153901.png](/img/assets/Registered%20Customer%20Information%20-%20image-20260426-153901.png)

The following table describes the features that support common follow-up tasks.

|**Callout ID**|**Feature**|**Description**|
|-|-|-|
|1|**Customer Name**|Display the name of the customer associated with the order.|
|2|**Customer ID**|Display the unique identifier assigned to the customer.|
|3|**Phone Number**|Display the contact number of the customer.|
|4|**Email ID**|Display the email address of the customer for communication.|
