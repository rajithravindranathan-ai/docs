# Managing Order Items and Actions

### Introduction

This section provides a centralized view of item-level details for the selected order. It allows users to review item information, track activity, and perform order-related actions through dedicated tabs such as Notes, Order Feed, and Price Adjustments.

---

### Order Items Section UI Reference Table

![image-20260426-151651.png](/img/assets/Managing%20Order%20Items%20and%20Actions%20-%20image-20260426-151651.png)

The following table describes the features that support common follow-up tasks.

|**Callout ID**|**Feature**|**Description**|
|-|-|-|
|1|**Order Summary**|Click **Order Summary** to view item-level details of the order.|
|2|**Notes**|Click **Notes** to view notes associated with the selected item.|
|3|**Order Feed**|Click **Order Feed** to view the activity history of the order.|
|4|**Price Adjustments**|Click **Price Adjustments** to view pricing changes applied to the order.|
