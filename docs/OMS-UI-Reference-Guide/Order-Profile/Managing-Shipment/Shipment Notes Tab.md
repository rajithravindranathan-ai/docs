# Shipment Notes Tab

## Introduction

The **Notes** tab in the Shipment Summary section displays notes associated with the shipment. It provides visibility into any additional information or remarks recorded for the shipment to support tracking and communication.

---

## Notes Behavior

![Shipment Notes Tab](/img/assets/Shipment%20Notes%20Tab%20-%20image-20260426-153234.png)

- Display all notes linked to the selected shipment.
- Show a **No Data Available** message when no notes are recorded.
- Update automatically when notes are added through the system.
