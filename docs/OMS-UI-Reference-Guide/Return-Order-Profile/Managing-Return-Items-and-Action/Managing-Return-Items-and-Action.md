# Managing Return Items and Action

### Introduction

The **Managing Return Items and Actions** section enables users to review returned items and perform necessary actions to complete the return process. It provides a structured view of return details along with access to supporting tabs for tracking activities, notes, and adjustments.

---

### Managing Return Items and Actions UI Reference

![Managing Return Items and Actions UI Reference](/img/assets/Managing%20Return%20Items%20and%20Action%20-%20image-20260424-113630.png)

| Callout ID | Field / Element | Description |
| --- | --- | --- |
| 1 | **Return Order Summary**| Displays item-level details of the return. |
| 2 | **Notes** | Allows users to view or add notes related to the return. |
| 3 | **Return Order Feed** | Displays a chronological log of return order item activities. |
| 4 | **Price Adjustments**| Displays any modifications made to the return amount. |
