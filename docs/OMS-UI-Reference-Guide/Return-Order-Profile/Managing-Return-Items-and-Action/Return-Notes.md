# Return Notes

The **Return Notes** section allows users to add and view comments related to a return order. It helps record important details, updates, or instructions during the return process.

---

![Return Notes](/img/assets/Return%20Notes%20-%20image-20260424-121010.png)

The **Notes** tab is positioned alongside **Return Order Summary**, **Return Order Feed**, and **Price Adjustments**. When selected, it displays all notes associated with the selected return order:

- If notes exist, they are listed in this section for quick reference.
- If no notes are available, the section displays **“No Data Available.”**

This allows users to review return-related context without navigating away from the return details.
