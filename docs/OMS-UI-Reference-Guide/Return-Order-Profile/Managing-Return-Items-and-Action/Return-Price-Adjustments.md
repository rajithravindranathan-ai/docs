# Return Price Adjustments

### Introduction

The **Price Adjustments** section allows users to view and track any manual or system-driven changes made to the return amount. It captures adjustment details such as when the change was made, who performed it, and the updated price.

---

### Price Adjustments UI Reference Table

![Return Price Adjustments](/img/assets/Return%20Price%20Adjustments%20-%20image-20260426-154556.png)

The following table describes the features that support common follow-up tasks.

| Callout ID | Field / Element | Description |
| --- | --- | --- |
| 1 | **Date** | Displays the date and time when the price adjustment was made. |
| 2 | **User** | Indicates the user who performed the price adjustment. |
| 3 | **Note** | Provides additional details or reason for the price adjustment. |
| 4 | **Adjusted Price** | Shows the updated price after applying the adjustment. |
