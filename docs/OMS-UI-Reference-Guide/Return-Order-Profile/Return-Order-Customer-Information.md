# Return Order Customer Information

### Introduction

The **Customer Information** displays the registered customer information associated with the selected return order. It provides visibility into customer identification and contact details required for order validation and communication.

---

### Customer Information UI Reference Table

![Return Order Customer Information](/img/assets/Return%20Order%20Customer%20Information%20-%20image-20260425-141250.png)

The following table describes the features that support common follow-up tasks.

| Callout ID | Feature | Description |
| --- | --- | --- |
| 1 | **Customer Name** | Displays the registered customer's name. |
| 2 | **Customer ID** | Displays the customer's unique identifier. |
| 3 | **Phone Number** | Displays the customer's contact number. |
| 4 | **Email ID** | Display’s the customer's email address. |
